FAQ CARPURIDE

Quel modèle pour ma moto ?

Les modèles universels (W702S Pro) s'adaptent à toutes les motos. Pour les BMW, choisissez la série BS qui intègre les commandes Wonder Wheel.

Le GPS fonctionne-t-il sans téléphone ?

Le W702 Plus a un GPS intégré avec Android 14, vous pouvez installer des cartes offline. Les autres modèles dépendent du téléphone.

Comment utiliser le code promo ?

Ajoutez le produit au panier sur carpuride.com, entrez le code LCDMH dans la zone prévue, la réduction s'applique immédiatement.

Le radar BSD est-il fiable ?

Oui, je l'ai testé sur autoroute et en ville, les alertes sont précises. C'est un vrai plus pour la sécurité.
