Questions fréquentes

FAQ AOOCCI

Quel code promo donne la plus grosse réduction ?

Essayez d'abord LCDMH28 qui donne -28%. Si ce code n'est pas actif, utilisez LCDMH25 (-25%) ou LCDMH (-22%).

Fonctionne-t-il vraiment sans internet ?

Oui, c'est son gros avantage. Il faut télécharger les cartes en Wi-Fi avant de partir, mais ensuite la navigation est 100% offline.

Peut-on utiliser Waze ?

Oui, via Android Auto ou CarPlay. Quand le téléphone est connecté, Waze s'affiche sur l'écran avec les alertes trafic.
