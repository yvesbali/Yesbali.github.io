L'Esprit du Parcours

Ce périple de 30 jours est une immersion totale dans les terres celtes. Tout commence par une remontée express vers le Nord pour franchir la Manche via le Shuttle. Le tracé nous emmène ensuite vers les Highlands sauvages, puis vers l'Irlande avant le retour. Le voyage, destination ou chemin ? Pour moi ce n'est pas le but qui compte, mais le chemin pour y arriver. Ce ruban d'asphalte, lien de vie entre les rencontres, les découvertes et les paysages. Un chemin parfois difficile, où les imprévus s'invitent sans prévenir, souvent surprenant, mais essentiel comme le sel de la vie.
